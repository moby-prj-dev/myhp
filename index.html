事業内容

Webアプリケーション開発を軸に、実装フェーズを中心としたコーディング業務をお引き受けします。

01 / Web Application

Webアプリ開発

Laravel / PHP を中心とした業務系Webアプリケーションの実装。データベース設計の落とし込みから、コントローラ・モデル・テスト実装までを担当します。

02 / Frontend Implementation

画面・UI実装

デザインカンプやワイヤーフレームから、Blade / Alpine.js / Livewire などを用いて画面を実装。レスポンシブとアクセシビリティに配慮した堅実な実装を行います。

03 / API Integration

外部API連携

決済・認証・SaaS・AI 等の外部APIとの連携実装。Webhook、リトライ制御、エラーハンドリング、レート制限対応など、運用に耐える実装を心がけます。

04 / Infrastructure

環境構築

Docker による開発環境の整備、Linux サーバへのデプロイ、CI まわりの構築など。小〜中規模の運用基盤を一通り立ち上げられます。

Profile

経歴・スキル

業務システム開発で積み上げた基礎の上に、Webアプリケーションの実装を重ねています。

約10年にわたり業務システムの開発に携わってきました。 C 言語・Visual Basic による業務アプリケーションや、 Baan ERP をはじめとする基幹システムの開発・保守を通じて、 「仕様に沿って、確実に動くコードを書く」という姿勢を身につけました。

近年は PHP / Laravel を中心としたWebアプリケーション開発に注力しており、 業務系で培った設計の読み解き方や、長く運用に耐えるコードを書く感覚を、Webの世界に持ち込んで仕事をしています。

得意領域は実装・コーディング。 提供された仕様・設計に対して、抜け漏れなく動くコードを書くこと、 そして次に触る人が読める形で残すことに重きを置いています。
